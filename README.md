MaskOn — Digital Sovereignty Secured

Powered by CHIRPS: Check, Inspect, Respond — Phone Security

MaskOn is a citizen‑centric mobile security app designed to protect users from surveillance, data leaks, and malicious tracking. It goes beyond traditional VPNs by continuously auditing apps, permissions, cookies, and network connections (LAN/WLAN).

Core Features
🔍 App Scanner: Detects hidden trackers and excessive permissions.

🧹 Cleaner: Clears cookies, cache, and resets identifiers.

🔐 Vault: Simplifies MFA and rotates passwords automatically.

📡 Network Audit: Monitors Wi‑Fi and SIM connections for privacy risks.

⚙️ CHIRPS Engine: Turns alerts into action through continuous inspection and response.
